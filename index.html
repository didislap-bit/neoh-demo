Ao enviar um pedido, o sistema exibe uma animação realista de envio WhatsApp com confirmação visual — ideal para demonstrações ao cliente.

O que o cliente vê: spinner "Enviando WhatsApp...", depois confirmação "✓ Mensagem enviada!" com o número do destinatário.

Quando quiser usar um número real, selecione UltraMsg ou Maytapi acima.
Como configurar o UltraMsg (5 minutos):
1. Acesse ultramsg.com → Create Account
2. Crie uma instância → escaneie o QR Code com seu WhatsApp
3. Copie o Instance ID e o Token da página da instância
Como configurar o Maytapi (5 minutos):
1. Acesse maytapi.com → Sign Up (3 dias grátis)
2. Crie um telefone → escaneie o QR Code com seu WhatsApp
3. Copie o Product ID, Phone ID e API Token
⚠️ Requer conta Meta Developers — pode levar alguns dias para aprovação. Use UltraMsg ou Maytapi enquanto isso.
